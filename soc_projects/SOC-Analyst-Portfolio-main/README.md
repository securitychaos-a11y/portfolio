# SOC Analyst Portfolio

## Overview

This repository documents my hands-on journey toward becoming a Security Operations Center (SOC) Analyst.

The portfolio is built around practical investigations performed in a personal home lab using Windows, Sysmon, Splunk Enterprise, Wazuh, Kali Linux, and Ubuntu.

Each project focuses on understanding attacker behavior, analyzing telemetry, building detections, and documenting investigations using industry-standard practices.
