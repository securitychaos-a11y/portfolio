# Wazuh-Based Windows Security Monitoring Lab

## Project Overview

This project focuses on designing and implementing a Security Information and Event Management (SIEM) lab using **Wazuh** to monitor and analyze security events generated by a Windows 11 system. The lab consists of an Ubuntu Server virtual machine running the Wazuh platform and a Windows 11 host configured with the Wazuh Agent. Security logs collected from the Windows system are forwarded to the Wazuh server for centralized monitoring, analysis, and alert generation.

The project provides hands-on experience in deploying an enterprise-grade security monitoring solution and understanding the workflow of a Security Operations Center (SOC).

---

## Objectives

- Deploy a Wazuh Server on Ubuntu Server.
- Configure the Wazuh Manager, Indexer, and Dashboard.
- Install and configure the Wazuh Agent on Windows 11.
- Collect Windows Event Logs in real time.
- Detect and analyze security events and suspicious activities.
- Visualize alerts using the Wazuh Dashboard.
- Gain practical SOC analyst experience with SIEM technologies.

---

## Workflow

1. The Windows 11 system generates security events.
2. The Wazuh Agent collects Windows Event Logs.
3. Logs are securely transmitted to the Wazuh Manager.
4. The Wazuh Manager analyzes events using built-in detection rules.
5. Relevant alerts are indexed and stored.
6. The Wazuh Dashboard visualizes alerts for monitoring and investigation.
